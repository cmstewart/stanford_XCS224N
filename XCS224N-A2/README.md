# Assignment 2: Understanding and Implementing Word2Vec

**Course:** XCS224N - Natural Language Processing with Deep Learning (Stanford, Spring 2026)

## Overview

This assignment dives into the mechanics of Word2Vec, implementing the skip-gram model with negative sampling from scratch. It builds on the embedding concepts from Assignment 1 by requiring a deeper understanding of the underlying math: cross-entropy loss, softmax, gradient derivations, and stochastic gradient descent.

## Topics Covered

- Softmax and cross-entropy loss for word prediction
- The skip-gram model: predicting context words from a center word
- Negative sampling as an efficient approximation to full softmax
- Deriving and implementing gradients for word vectors
- Stochastic gradient descent optimization
- Training word vectors on real text data

## Key Files

- `src/submission.py` - Implementation of Word2Vec with negative sampling
- `tex/submission.tex` - Written responses including gradient derivations (LaTeX)
