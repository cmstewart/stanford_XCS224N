# Assignment 1: Exploring Word Embeddings

**Course:** XCS224N - Natural Language Processing with Deep Learning (Stanford, Spring 2026)

## Overview

This assignment introduces word embeddings, the foundational representation that underlies nearly all modern NLP systems. It covers two approaches to building word vectors: count-based methods using co-occurrence matrices and prediction-based methods like Word2Vec.

## Topics Covered

- Co-occurrence matrices and their properties (symmetry, sparsity, context windows)
- Dimensionality reduction via Truncated SVD to produce dense word vectors
- Cosine similarity for measuring word relatedness
- Word analogies and the relational structure captured by embeddings
- Bias in word vectors and its implications for downstream models
- Implementing word feature extraction and exploring pre-trained GloVe embeddings
